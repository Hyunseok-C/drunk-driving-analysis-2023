### 🎯 가설 설정
1. 평일보다 주말에 음주운전 단속 적발 건수가 많을 것이다.
2. 젊은 남성이 음주운전 단속에서 가장 많이 적발될 것이다.
3. 겨울철(연말) 밤 시간대에 음주운전 단속 적발 건수가 가장 많을 것이다.
4. 음주운전 단속 적발 건수가 많을수록 사고 발생 건수와 사상자 수가 높아질 것이다.
5. 서울시 유흥가에서 음주운전 단속 적발과 사고 발생 건수가 많을 것이다.

### 🧠 분석 결과
1. `가설1` 금요일과 주말(특히 토요일)에 음주운전 적발 건수가 뚜렷하게 많았다.
2. `가설2` 20~40대 남성이 전체 적발자의 대부분을 차지하였다.
3. `가설3` 예상과 달리 겨울보다 여름철(6-9월)에 적발이 집중되었으며, 밤 10시~새벽 2시 사이에 가장 빈번했다.
4. `가설4` 자치구별로 적발 건수가 많을수록 사고 발생 및 사상자 수가 함께 증가하는 경향을 보였다.
5. `가설5` 강남구·송파구·마포구 등 유흥 밀집 지역과 교차로 주변에서 적발과 사고가 집중되었다.
